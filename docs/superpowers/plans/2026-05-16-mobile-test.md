# Mobile Test Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Create a parallel Android-first mobile test app from the current working web app without breaking the live GitHub Pages version.

**Architecture:** Keep the existing static HTML/CSS/JS app as the source of truth. Add a minimal Node/Capacitor wrapper that copies the web files into a native Android shell while preserving the current Supabase-backed data model.

**Tech Stack:** Static HTML/CSS/JavaScript, Supabase, GitHub Pages, Node/npm, Capacitor, Android Studio/Gradle.

---

## File Structure

- Modify: `package.json` - define local scripts and Capacitor dependencies.
- Create: `capacitor.config.json` - Capacitor app metadata and web directory.
- Create: `www/` - generated/static mobile web bundle copied from the current root files.
- Create: `manifest.webmanifest` - PWA metadata for installability and future store polish.
- Create: `assets/` - app icons/splash source assets when available.
- Modify: `index.html` - link manifest and mobile metadata.
- Modify: `README.md` - document mobile test workflow.
- Create after Capacitor init: `android/` - native Android project generated by Capacitor.

## Task 1: Prepare Static App For Mobile Packaging

- [x] **Step 1: Add a minimal `package.json`**

Create `package.json` with scripts that preserve the static app:

```json
{
  "name": "equilibrador-5v5",
  "version": "0.1.0",
  "private": true,
  "scripts": {
    "dev": "node server.mjs 5173",
    "check": "node --check app.js",
    "build:web": "node scripts/build-web.mjs",
    "cap:sync": "npm run build:web && npx cap sync",
    "cap:open:android": "npx cap open android"
  },
  "dependencies": {
    "@capacitor/android": "^7.0.0",
    "@capacitor/cli": "^7.0.0",
    "@capacitor/core": "^7.0.0"
  }
}
```

- [x] **Step 2: Create `scripts/build-web.mjs`**

The build script should copy the static files into `www/`:

```js
import { cp, mkdir, rm } from "node:fs/promises";

const files = [
  "index.html",
  "styles.css",
  "app.js",
  "app-config.js",
  "manifest.webmanifest"
];

await rm("www", { recursive: true, force: true });
await mkdir("www", { recursive: true });

for (const file of files) {
  await cp(file, `www/${file}`, { recursive: true });
}

await cp("assets", "www/assets", { recursive: true, force: true }).catch(() => {});
```

- [x] **Step 3: Add `manifest.webmanifest`**

Use a neutral app identity for the test:

```json
{
  "name": "Equilibrador 5v5",
  "short_name": "5v5",
  "start_url": ".",
  "display": "standalone",
  "background_color": "#eef5f1",
  "theme_color": "#1f7a4d",
  "icons": []
}
```

- [x] **Step 4: Link the manifest in `index.html`**

Add inside `<head>`:

```html
<link rel="manifest" href="manifest.webmanifest">
<meta name="theme-color" content="#1f7a4d">
<meta name="apple-mobile-web-app-capable" content="yes">
<meta name="apple-mobile-web-app-title" content="Equilibrador 5v5">
```

- [x] **Step 5: Verify the web app still parses**

Run:

```bash
npm run check
```

Expected: no syntax errors.

## Task 2: Add Capacitor Android Shell

- [x] **Step 1: Install dependencies**

Run:

```bash
npm install
```

Expected: `node_modules/` and `package-lock.json` are created.

- [x] **Step 2: Add Capacitor config**

Create `capacitor.config.json`:

```json
{
  "appId": "com.kremaster.equilibrador5v5",
  "appName": "Equilibrador 5v5",
  "webDir": "www",
  "server": {
    "androidScheme": "https"
  }
}
```

- [x] **Step 3: Build web bundle**

Run:

```bash
npm run build:web
```

Expected: `www/index.html`, `www/app.js`, `www/styles.css`, and `www/app-config.js` exist.

- [x] **Step 4: Add Android project**

Run:

```bash
npx cap add android
```

Expected: `android/` directory is created.

- [x] **Step 5: Sync web app into Android**

Run:

```bash
npm run cap:sync
```

Expected: Capacitor copies the `www/` bundle into Android assets.
